Images

All images for the web application should be stored in the img/ directory.

Icons

This template includes Zondicons, a free SVG icon library with 270 icons. These can be included in the HTML template files with @include(<relative_path_to_svg>) and it the svg file will be included inline so standard utility classes from Tailwind can be applied. It is best to wrap the icons in a wrapper containing the icon class, as this will default to the icon being the size and color of the font.
